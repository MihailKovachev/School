1. Същност - йоннообменни процеси (ЙОП), при които зарядът на йоните не се променя
	
	**а) условие за протичане на реакцията** - поне два от йоните трябва да напуснат разтвора под формата на утайка, газ или слаб електролит, т.е. поне един от продуктите трябва да е неразтворим
	
	$\ce{Na+NO3^- + H^+Cl^- -> X}$ - не протича реакция, защото $\ce{Na^+Cl^-}$ и $\ce{HNO3^-}$ са разтворими
	
	$\ce{AgNO3 + HCl -> } \text{бяла утайка}$, защото $\ce{Ag^+Cl^-}$ е неразтворим 
	

2. Изразяване
	
	**а) общо уравнение**
	
	$$\ce{Ag^+NO3^- + H^+Cl^- -> Ag^+Cl^- + H^+NO3^-}$$
	
	**б) пълно йонно уравнение** - всички разтворими вещества се записват във вид на йони
	
	$$\underbrace{\ce{Ag^+ + NO3^}} + \underbrace{\ce{H^+ + Cl^-}} \rightarrow \ce{Ag^+Cl^-} + \underbrace{\ce{H^+ + NO3^-}}$$
	
	**в) съкратено йонно уравнение** - записват се само йоните, напускащи системата
	
	$$\ce{Ag^+ + Cl^- -> Ag^+Cl^-}$$

3. Видове йоннообменни процеси
	
	**а) с образуване на утайка**
	
	$$\ce{Ba^2+Cl2^- + H2^+SO4^2- -> Ba^2+SO4^2- + 2H^+Cl^-}$$
	
	$$\underbrace{\ce{Ba^2+ + 2Cl^-}} + \underbrace{\ce{2H^+ + SO4^2-}} \rightarrow \ce{Ba^2+SO4^2-} + \underbrace{\ce{2H^+ + 2Cl^-}}$$
	
	$$\ce{Ba^2+ + SO4^2- -> Ba^2+SO4^2-}$$
	
	**б) с отделяне на газ**
	
	$$\ce{Na2^+CO3^2- + 2H^+NO3^- -> 2Na^+NO3^- + H2^+CO3^2-}$$
	
	$$\underbrace{\ce{2Na^+ + CO3^2-}} + \underbrace{\ce{2H^+ +2NO3^2-}} \rightarrow \underbrace{\ce{2Na^+ + 2NO3^-}} + \ce{H2^+CO3^2-}$$
	
	$$\ce{CO3^2- + 2H^+ -> H2^+CO3^2-}$$
	
	**в) с образуване на слаб електролит** - най-често е неутрализация
	
	$$\ce{Na^+OH^- + H^+Cl^- -> Na^+Cl^- + H2O}$$
	
	$$\underbrace{\ce{Na^+ + OH^-}} + \underbrace{\ce{H^+ + Cl^-}} \rightarrow \underbrace{\ce{Na^+ + Cl^-}} + \ce{H2O}$$
	
	$$\ce{H^+ + OH^- -> H2O}$$
