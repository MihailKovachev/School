1. Киселини - вещества, които се дисоциира само до $\ce{H^+}$ катиони и киселинни аниони
	
	$$\ce{H_nA -> nH^+ + A^n-}$$
	
	**а) протонност на киселините** - броят молове $\ce{H^+}$ катиони при дисоциация ( $n$ )
	- монопротонни - $n = 1$: $\ce{HCl}$
	- полипротонни - $n \gt 1$: $\ce{H2SO4, H3PO4}$ - степенна дисоциация
	
	$$\ce{H2SO4 -> H^+ + H+SO4^-}$$
	
	$$\ce{HSO4^- <=> H^+ + SO4^2-}$$
	
	$$\ce{H2SO4 -> 2H^+ + SO4^2-}$$

2. Основи - вещества, които се дисоциира само до $\ce{OH^-}$ аниони и метални катиони
	
	$$\ce{M(OH)_m -> M^m+ + mOH^-}$$
	
	**а) валентност на основите** - броят молове $\ce{OH^-}$ аниони при дисоциация ( $m$ )
	- моновалентни - $m = 1$: $\ce{NaOH, KOH}$
	- поливалентни - $m \gt 1$: $\ce{Ca(OH)2}$ - степенна дисоциация
	
	$$\ce{Ca(OH)2 -> Ca^+OH^- + OH^-}$$
	
	$$\ce{CaOH -> Ca^2+ + OH^-}$$
	
	$$\ce{Ca(OH)2 -> Ca^2+ +2OH-}$$

3. Соли - вещества, които са продукти на процеса неутрализация
	
	**а) нормални соли** - продукти на пълна неутрализация ($H$ в едното се изравнява с $OH$ в другото)
	- дисоциират се до метални (амониеви) катиони и киселинни аниони
	
	$$\ce{Al(OH)3 + 3HNO3 -> Al(NO3)3 + 3H2O}$$
	
	$$\ce{Al(NO3)3 -> Al^3+ + 3NO3^-}$$
	
	**б) хидрогенсоли (киселинни соли)** - продукти на непълна неутрализация на полипротонни киселини
	- степенна дисоциация до метални (амониеви) катиони, киселинни аниони и водородни катиони
	- именуват се като към името на аниона се добави "хидроген-", напр. "натриев хидрогенкарбонат"
	
	$$\ce{NaHCO3 -> Na^+ +H^+CO3^-}$$
	
	$$\ce{HCO3 <=> H+ + CO3^2-}$$
	
	$$\ce{NaHCO3 -> Na^+ + CO3^2- + H^+}$$
	
	**в) хидроксисоли (основни соли)** - продукти на непълна неутрализация на поливалентни основи
	- степенна дисоциация до метални катиони, киселинни аниони и хидроксидни аниони
	- именуват се като към името на аниони се добави представката "хидрокси-", напр. "калциев хидроксинитрат"
	
	$$\ce{Ca(OH)NO3 -> Ca(OH)^+ + NO3^-}$$
	
	$$\ce{Ca(OH)^+ <=> Ca^2+ + OH^-}$$
	
	$$\ce{Ca(OH)NO3 -> Ca^2+ + OH^- + NO3^-}$$
	
	**г) двойни соли** - продукти на съвместна неутрализация (между две киселини и една основа или между две основи и една киселина)
	- дисоциират се до два вида метални (амониеви) катиони и киселинен анион
	
	$$\ce{KAl(SO4)2 -> K^+ + Al^3+ + 2SO4^2-}$$
