1. Алкални метали - $\ce{Li, Na, K, Rb, Cs}$

2. Физични свойства
	- сребристобял цвят (само цезият е златист)
	- метален блясък
	- меки и леки
	- ниска температура на топене
	- висока топло- и електропроводимост
	- най-активните метали
	- съхраняват се в инертна среда

3. Кислородосъдържащи съединения
	
	**а) видове**
	- оксиди - съдържат оксидни ($\text{O}^{2-}$) йони
	
	$$\text{M}_2^{+}\ce{O^{2-}}$$
	
	- пероксиди - съдържат пероксидни ($\text{O}_2^{2-}$) йони
	
	$$\text{M}_2^{+}\text{O}_2^{2-}$$
	
	- супероксиди - съдържат супероксидни ($\text{O}_2^{-}$) йони
	$$\text{M}^{+}\text{O}_2^{-}$$
	
	**б) получаване**
	- горене на алкален метал на въздух
	
	$$\ce{4Li + O2 -> 2Li_2O}, \text{ в малки количества }\ce{2Li + O2 -> Li2O2}$$
	
	$$\ce{2Na + O2 -> Na2O2}, \text{ в малки количества } \ce{4Na + 2O2 -> Na2O }$$
	$$\ce{K, Rb, Cs + O2 -> MO2 } \text{ (супероксиди) }$$
	
	- горене на алкален метал в чист кислород
	
	$$\ce{4Na + O2 -> 2Na2O} \text{ (недостиг на кислород) }$$
	
	$$\ce{2Na + O2 ->[Na2O] Na2O2} \text{ (излишък на кислород)}$$
	
	$$\ce{2K + O2 -> K2O2} \text{ (недостиг на кислород
	)}$$
	
	$$\ce{K + O2 ->[K2O2] KO2 } \text{ (излишък на кислород) }$$
	
	- стапяне на метала с негово съединение - за получаване на алкални оксиди
	
	$$\ce{M2O2 + 2M ->[t°] 2M2O}$$
	
	$$\ce{6MNO3 + 10M ->[t°] 6M2O + 3N2}$$
	
	**в) химични свойства**
	- на оксидите 
	
	|Взаимодействие|Продукти|Пример|
	|:--:|:--:|:--:|
	|с вода|хидроксид|$\ce{M2O + H2O -> 2MOH}$|
	|с киселини|сол и вода|$\ce{M2O + 2HCl -> 2MCl + H2O}$|
	|с киселинни оксиди|сол|$\ce{Na2O + SO2 -> Na2SO3}$|
	
	- на пероксидите
	
	|Взаимодействие|Продукти|Реакция|
	|:--:|:--:|:--:|
	|с вода|основа и водороден пероксид|$\ce{M2O2 + 2H2O -> 2MOH + H2O2}$|
	|с киселина|сол и водороден пероксид||
	
	**г) употреба** - нямат значителна промишлена употреба

4. Хидроксиди ($\ce{OH-}$)
	
	**а) получаване**
	
	|Реагенти|Продукти|Реакция|
	|:--:|:--:|:--:|
	|алкален метал + $\ce{H2O}$|алкален хидроксид + $\ce{H2}$|$\ce{2M + 2H2O -> 2MOH + H2_{(g)} ^}$|
	|алкален оксид + $\ce{H2O}$|алкален хидроксид|$\ce{M2O + H2O -> 2MOH}, \Delta H \ll 0$|
	|алкален пероксид + $\ce{H2O}$|алкален хидроксид + $\ce{H2O2}$|$\ce{M2O2 + 2H2O -> 2MOH + H2O2}$|
	|алкален супероксид + $\ce{H2O}$|алкален хидроксид + $\ce{O2 + H2O2}$|$\ce{2MO2 + 2H2O -> MOH + O2 + H2O2}$|
	
	- хлоро-алкална електролиза
	
	$$\ce{2NaCl_{(aq)} + 2H2O_{(l)} ->[electrolysis] 2NaOH_{(aq)} + H2_{(g)} + Cl2_{(g)}}$$
	
	**б) химични свойства** - силно основни
	- огромна разтворимост във вода
	
	|Взаимодействие|Продукти|Реакция|
	|:--:|:--:|:--:|
	|с киселини|сол и вода||
	
	**в) употреба**
	- $\ce{NaOH}$ (сода каустик) - производство на сапуни, лекарства, отпушване на канали

5. Карбонати ($\text{CO}_3^{2-}$) и хидрогенкарбонати ($\text{HCO}_3^-$)
	
	**а) получаване**
	- на карбонати
	
	|Реагенти|Продукти|Реакция|
	|:--:|:--:|:--:|
	|алкален оксид + $\ce{CO2}$|карбонат|$\ce{M2O_{(s)} + CO2_{(g)} -> M2CO3_{(g)}}$|
	|алкален пероксид + $\ce{CO2}$|карбонат + $\ce{O2}$|$\ce{2M2O2 + 2CO2 -> 2M2CO3 + O2}$|
	|алкален хидроксид + $\ce{CO2}$|карбонат + $\ce{H2O}$|$\ce{2MOH_{(s)} + CO2 -> M2CO3 + H2O}$|
	|алкален хидрогенкарбонат|карбонат + $\ce{CO2 + H2O}$|$\ce{2MHCO3_{(s)} ->[t°] M2CO3_{(s)} + CO2_{(g)} + H2O_{(g)}}$|
	
	- на хидрогенкарбонати
	
	|Реагенти|Продукти|Реакция|
	|:--:|:--:|:--:|
	|алкален карбонат + $\ce{ H2O + CO2}$|хидрогенкарбонат|$\ce{M2CO3_{(s)} + H2O_{(l)} + CO2_{(g)} -> 2MHCO3_{(s)}}$|
	|разтвор на алкална основа + $\ce{CO2}$|хидрогенкарбонат|$\ce{2MOH_{(aq)} + CO2_{(g)}-> MHCO3_{(aq)}}$|
	
	**б) химични свойства**
	- на карбонатите - основни
	
	$$\ce{CO3^{2-} + H2O <=> HCO3- + OH-}$$
	
	|Взаимодействие|Продукти|Реакция|
	|:--:|:--:|:--:|
	|със силна киселина ("шумяща проба")|сол, $\ce{CO2, H2O}$|$\ce{I: CO3^{2-} + H3O+ -> HCO3- + H2O}$ <br/> $\ce{II: \ce{HCO3- + H3O+ -> CO2 + 2H2O}}$|
	
	- на хидрогенкарбонатите
	
	|Взаимодействие|Продукти|Реакция|
	|:--:|:--:|:--:|
	|термично разлагане|карбонат, $\ce{CO2, H2O}$|$\ce{2MHCO3 ->[t°] K2CO3 + CO2_{(g)} + H2O_{(l)}}$|
	
	**в) приложение**
	- $\ce{Na2CO3}$ (калцинирана сода) - производство на стъкло, хартия, сапуни
	- $\ce{NaHCO3}$ (сода бикарбонат) - в кулинарията
	
	$$\ce{2NaHCO3 ->[t°] Na2CO3 + CO2 + H2O}$$
	
	$$\ce{NaHCO3 +}\underset{\text{слаба киселина}}{\ce{H+}} \ce{-> Na+ + CO2 + H2O}$$

6. Халогениди
	
	**а) получаване**
	- директно взаимодействие
	
	$$\ce{2M + X2 -> 2MX}$$
	
	- взаимодействие с халогеноводород
	
	$$\ce{2M + 2HX -> 2MX + H2_{(g)} ^}$$
	
	- чрез обменна реакция между алкален карбонат и халогеноводород
	
	$$\ce{M2CO3 + 2HX -> 2MX + CO2_{(g)} + H2O_{(l)}}$$
	
	- чрез обменна реакция между алкален хидроксид и халогеноводород
	
	$$\ce{MOH + HX -> MX + H2O}$$
	
	**б) химични свойства** - соли с йонно-кристален строеж, често под формата на кристалохидрати
	
	**в) приложение**
	- $\ce{NaCl}$ (готварска сол) - хранително-вкусова промишленост, производство на $\ce{Na, Cl, NaOH, HCl}$

7. Хидриди
	
	**а) получаване** - директно взаимодействие между алкален метал и водород
	
	$$\ce{2M + H2 ->[t°] 2MH}$$
	
	**б) химични свойства** - изключително силни основни свойства заради хидридния ($\ce{H-}$) йон
	
	|Взаимодействие|Продукти|Реакция|
	|:--:|:--:|:--:|
	|с вода|алкална основа и водород|$\ce{MH + H2O_{(l)} -> MOH_{(aq)} + H2_{(g)} ^}$|
	
	**в) приложение**
	- бърз и лесен начин за получаване на водород
	- надуваеми спасителни жилетки
	- ракетно гориво
